## 1. 产品概述

虚拟戏服试穿工具是一款面向剧团服装组和演员的在线造型预览应用，解决传统试装耗时耗力的痛点。用户上传正面半身照后，系统自动识别关键部位，可选择30+种经典戏曲行头和15+种盔头进行虚拟试穿，支持位置微调以确保贴合度，并可保存搭配方案生成对比图供导演挑选。

- 核心目标：将传统耗时的实体试装流程数字化，提升排戏效率
- 目标用户：剧团服装师、演员、导演
- 核心价值：在线预览造型效果，大幅减少实际换装时间

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地使用 | 上传照片、试穿服装、保存方案、生成对比图 |

### 2.2 功能模块
1. **主编辑页面**：照片上传/拍摄、Canvas画布、服装选择面板、微调控制器
2. **方案管理面板**：已保存搭配方案列表、加载/删除功能
3. **对比图生成器**：选择3套方案生成1000×1200像素对比图

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主编辑页 | 照片上传模块 | 支持本地上传或摄像头拍摄正面半身照，自动识别面部和肩部轮廓 |
| 主编辑页 | Canvas编辑器 | 实时显示人物照片、服装图层、盔头图层，支持拖拽缩放 |
| 主编辑页 | 服装选择面板 | 分类展示蟒袍、靠、帔、褶子、官衣五大类，每类含多种颜色纹样 |
| 主编辑页 | 盔头选择面板 | 展示王帽、相貂、纱帽、凤冠、紫金冠等盔头选项 |
| 主编辑页 | 微调控制器 | 八向方向键移动、缩放滑块控制、重置按钮 |
| 主编辑页 | 方案保存模块 | 命名并保存当前搭配，最多存储20套本地方案 |
| 方案管理页 | 方案列表 | 展示缩略图和名称，支持加载/删除操作 |
| 对比图生成页 | 方案选择器 | 勾选3套方案，一键生成横向或纵向拼接对比图 |

## 3. 核心流程

用户上传或拍摄正面半身照 → 系统自动识别面部和肩部关键点 → 从服装库选择行头类别和具体款式 → 服装自动贴合到人物身上 → 通过控制器微调位置和缩放确保领口对齐 → 可选戴盔头并调整 → 保存为搭配方案 → 选择3套方案生成对比图

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深红 (#8B0000) 搭配金色 (#D4AF37) 点缀，体现戏曲传统美学
- **辅助色**：米白 (#FFF8E7) 背景，深灰 (#2D2D2D) 文字
- **按钮风格**：圆角矩形，传统戏曲纹样装饰边框，悬停有金色渐变效果
- **字体**：标题使用书法风格衬线字体，正文使用清晰易读的无衬线字体
- **布局**：三栏式布局，左侧为素材选择，中间为Canvas编辑区，右侧为方案和控制面板
- **视觉元素**：融入传统云纹、回纹等装饰元素，木质纹理背景

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主编辑页 | 顶部导航 | Logo、项目名称、帮助按钮、对比图生成入口 |
| 主编辑页 | 左侧面板 | 分类标签页（服装/盔头）、网格布局素材卡片、滚动容器 |
| 主编辑页 | 中间区域 | Canvas画布（带参考线）、照片上传按钮、重置按钮 |
| 主编辑页 | 右侧面板 | 微调控制器（方向键+滑块）、当前选中信息、保存按钮 |
| 方案管理页 | 方案网格 | 卡片式展示，悬停显示操作按钮，数量指示器 |
| 对比图生成页 | 选择区域 | 复选框+缩略图，生成按钮，预览区域，下载按钮 |

### 4.3 响应性
- 桌面端优先设计（1280px以上）
- 平板端（768-1279px）：左右面板可折叠收起，使用抽屉式菜单
- 移动端（768px以下）：单栏布局，垂直堆叠，Canvas区域最大显示

### 4.4 交互细节
- 服装卡片悬停放大并显示详细信息
- 拖拽服装到Canvas时显示半透明预览
- 微调操作有实时视觉反馈
- 保存/加载操作有成功/失败动画提示
